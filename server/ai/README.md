# server/ai/

LLM 调用模块，支持 OpenAI / Anthropic / 自定义接口。

## 文件

| 文件 | 地位 | 功能 |
|------|------|------|
| generate.js | 核心 | 调用 LLM 生成测试用例、测试报告、需求分析，解析返回的 JSON（自动处理各种格式） |

## 需求分析功能

- 接口：`POST /api/generate-cases`，`mode: analyze`
- 返回：分类的问题列表（边界未定义、逻辑漏洞、歧义描述、遗漏场景、技术风险、体验问题）
- 前端：内联显示分析报告，支持一键复制

## JSON 解析增强

自动处理 LLM 返回的各种格式：
- 纯 JSON 数组
- JSON 对象（自动提取其中的数组）
- markdown 代码块包裹的 JSON（```json ... ```）
- 被转义的 JSON 字符串
- 中文引号自动转换为标准 JSON 引号
- 未转义的换行符修复
- 尾部逗号修复

解析失败时会自动重试一次，最终失败则返回原始文本。
