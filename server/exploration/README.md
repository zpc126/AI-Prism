# Exploration

本目录只处理用户主动发起的 Web AI 探索，不改变普通用例“脚本优先、失败不自动回 Agent”的执行边界。默认按动作数停止；持续模式不限制动作，可选 1 至 1440 分钟时长，留空则由用户手动停止。登录凭据仅用于本地前置，模型提示词和持久化记录都不包含密码。

| 文件 | 地位 | 功能 |
| --- | --- | --- |
| `routes.js` | HTTP 入口 | 规范运行参数、启停 SSE、查询历史，并优先读取 `server/data/screenshots`、兼容根目录旧证据 |
| `runner.js` | 核心执行器 | 同源/只读保护、本地登录、动作或时长停止、Agent 探索和结果解析 |
| `store.js` | 数据层 | 迁移旧表并保存探索范围、运行策略、问题、动作和证据 |
| `runner.test.js` | 单元测试 | 验证 URL、安全动作、可选时长和结构化结果解析 |
