# src/

前端界面，渲染进程的代码和资源。

## 文件

| 文件 | 地位 | 功能 |
|------|------|------|
| index.html | 页面 | Prism 主界面结构、底部数据统计、供应链登录示例、历史分析报告入口与产品引导 |
| welcome.js | 引导 | 首次使用引导、Prism 小图标和 API Key 配置提示 |
| reports.html | 页面 | 独立自动化测试报告列表、执行详情、旧报告步骤回填与截图预览 |
| renderer.js | 逻辑 | 识别产品一级标题、首页统计、用例库版本编辑、历史分析报告标题编辑、分享链接复制、对话恢复、分阶段等待反馈、图标点击与导出 |
| canvas.js | 画布 | 思维导图布局、渲染、节点置顶交互与阶段进度展示 |
| brain.js | 大脑 | 知识图谱、碎片管理、可滚动 Flipbook 详情卡片 |
| eval.js | 评估 | 评估集管理、运行监控、报告展示 |
| settings.js | 设置 | 通过 Web API 管理模型与 API Key 配置 |
| styles/ | 样式 | CSS 样式文件 |

## 动效系统

| 功能 | 说明 |
|------|------|
| 思考过程 | 后端 LLM 分析需求，返回核心模块，前端实时展示 |
| 画布过渡 | 从对话区平滑跳转到画布 |
| 用例聚焦 | 新生成的用例会聚焦高亮显示 |
| 灵动岛 | 与画布对话条分离，互不遮挡 |
| Flipbook | 点击知识节点后原地展开可滚动详情卡片，带缩放动画 |
