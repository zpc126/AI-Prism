# src/

前端界面，渲染进程的代码和资源。

## 文件

| 文件 | 地位 | 功能 |
|------|------|------|
| index.html | 页面 | Prism 主界面结构、顶部 Web 探索入口、底部统计、提 Bug 工作台、防误触用例入口与脚本加载 |
| welcome.js | 引导 | 首次使用引导、Prism 小图标和 API Key 配置提示 |
| reports.html | 页面 | 独立自动化测试报告列表、执行详情、旧报告步骤回填与截图预览 |
| renderer.js | 逻辑 | 首页和用例主流程、GitLab Bug 编辑与里程碑选择、探索问题草稿与截图证据接入、历史分析报告、对话恢复和导出 |
| exploration.js | 工作台 | Web AI 限次/持续探索、可选最长时间、SSE 过程、覆盖/问题/历史和一键转 Bug |
| canvas.js | 画布 | 思维导图布局、渲染、用例操作栏、防单击误执行与阶段进度展示 |
| brain.js | 大脑 | 知识图谱、碎片管理、可滚动 Flipbook 详情卡片 |
| eval.js | 评估 | 评估集管理、运行监控、报告展示 |
| settings.js | 设置 | 通过 Web API 管理模型与 API Key 配置 |
| script-library.js | 组件 | 自动化脚本库工作台、旧脚本兼容编辑、脚本执行、脚本包导出和 GitLab 提交 |
| styles/ | 样式 | 主界面、探索工作台和业务组件样式 |

## 动效系统

| 功能 | 说明 |
|------|------|
| 思考过程 | 后端 LLM 分析需求，返回核心模块，前端实时展示 |
| 画布过渡 | 从对话区平滑跳转到画布 |
| 用例聚焦 | 新生成的用例会聚焦高亮显示 |
| 灵动岛 | 与画布对话条分离，互不遮挡 |
| Flipbook | 点击知识节点后原地展开可滚动详情卡片，带缩放动画 |
